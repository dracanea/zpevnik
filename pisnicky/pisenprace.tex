}

\vers{1}{
\chord{G}Ráno vstávám, \chord{D}hledám práci, \chord{F}práce není, \chord{C}není práce,\\
\chord{E\bb}není práce \chord{G}pro mé ruce, \chord{Am}po mé pravé \chord{D}ruce káva.
}

\vers{2}{
Nese kávu ruka k ústům, ke kterým má zjevně úctu,\\
ústa sají ranní kávu, je to práce pro zábavu.
} \refsm{}

\vers{3}{
Práce pláče, práce volá, rozplývá se v širá kola,\\
kola kroutí v košík proutí, až si počkám, tak se dočkám.
}

\vers{4}{
Pro mé kvítí slunce svítí, bez té práce na koláče,\\
ústa hnětou měkké těsto, já vám jejich těžkou práci (dám.)
} \refsm{} $\times 2$
