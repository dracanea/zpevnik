\hyt{velicenstvokat}
\song{Veličenstvo kat}

\vers{1}{
\chord{Dm}V ponurém osvětlení \chord{C}gotického \chord{Dm}sálu, \chord{F}kupčíci vyděšení \chord{Gm}hledí do mi\chord{A}sálů\\
\chord{Dm}a houfec \chord{B}mordýřů si \chord{C}žádá požeh\chord{F}ná\chord{C}ní, \rep{\chord{Gm}vždyť první z \chord{Dm}rytířů je \chord{A\7}Veličenstvo \chord{Dm}kat.}
}

\vers{2}{
Kněz-ďábel, co mši slouží, z oprátky má štolu, pod fialovou komží láhev vitriolu.\\
Pach síry z hmoždířů se valí k rudé kápi \rep{prvního z rytířů, hle, Veličenstvo kat.}
}

\refrainn{1}{
\chord{F}Na korouhvi \chord{C}státu \chord{B}je emblém s \chord{C}gilotinou,\\
\chord{F}z ostnatýho \chord{C}drátu \chord{B}páchne to shnilotinou.\\
\chord{Gm}V kraji hnízdí hejno krkav\chord{Dm}čí,\\
\chord{Gm}lidu vládne mistr poprav\chord{A}čí.
}

\vers{3}{
Král klečí před Satanem, na žezlo se těší a lůza pod platanem radu moudrých věší.\\
A zástup kacířů se raduje a jásá, \rep{vždyť prvním z rytířů je Veličenstvo kat.}
}

\vers{4}{
Na rohu ulice vrah o morálce káže, před vraty věznice se procházejí stráže,\\
z vojenských pancířů vstříc černý nápis hlásá, \hspace{-7pt}\rep{\hspace{-5pt}že prvním z rytířů je Veličenstvo kat.\hspace{-10pt}}
}

\refrainn{2}{
Nad palácem vlády ční prapor s gilotinou,\\
děti mají rády kornouty se zmrzlinou.\\
Soudcové se na ně zlobili,\\
zmrzlináře dětem zabili.
}

\vers{5}{
Byl hrozný tento stát, když musel jsi se dívat, jak zakázali psát a zakázali zpívat.\\
A bylo jim to málo, poručili dětem \rep{modlit se, jak si přálo Veličenstvo kat.}
}

\vers{6}{
S úšklebkem Ďábel viděl pro každého podíl, syn otce nenáviděl, bratr bratru škodil.\\
Jen motýl smrtihlav se nad tou zemí vznáší, \rep{kde v kruhu tupých hlav dlí Veličenstvo kat.}
}
