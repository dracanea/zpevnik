\song{Válka růží} \interpret{spiritualkvintet}{Spirituál kvintet}
\vers{1}{
Už \chord{Dm}rozplynul se \chord{G}hustý dým, \chord{Dm}derry down, hey, \chord{A}down, hey down,\\
nad \chord{Dm}ztichlým polem \chord{Gm}válečným, derry \chord{Dm}down, \chord{A}\\

je válka růží, derry derry derry down, hey down.
}

\vers{3}{

}

\vers{4}{

je válka růží, derry derry derry down, hey down.
}
