\hyt{velkyvuz}
\song{Velký vůz}

\intro{
\chord{Em}Jít tmou může, kdo hvězdy zná\dots
}

\refrain{
\chord{Em}Běž \chord{A}dál, teď \chord{Em}nesmíš to vzdát, \chord{Em}běž \chord{A}dál, svý \chord{Em}štěstí zkus.\\
Pro ty, \chord{G}kteří táhnou \chord{D}na sever jak \chord{Em}hejna nočních \chord{Hm}ptáků \chord{Em}tmou svítí \chord{Hm}Velký \chord{Em}vůz.
}

\vers{1}{
Tvoji \chord{Em}dlouhou pouť přetne kalná řeka, běž dál a svý štěstí zkus.\\
Tam \chord{G}za vodou je \chord{D}svoboda, tam \chord{Em}starý muž tě \chord{Hm}čeká, \chord{Em}tmou svítí \chord{Hm}Velký \chord{Em}vůz
} \refsm{}

\vers{2}{
Náhle končí cesta, dál je jen mrtvá pláň, běž dál, svý štěstí zkus.\\
Močál čeká na chybu, ten močál chce svou daň, tmou svítí velký vůz.
} \refsm{}

\vers{3}{
Než se vzbudí den, musíš provést svůj plán, běž dál, svý štěstí zkus.\\
Bílé ta noc ošálí, černým znamení dal Pán, tmou svítí Velký vůz.
} \refsm{}

\vers{4}{
Klesáš únavou, vždyť chvíli počkat ses bál, běž dál, svý štěstí zkus.\\
Levá, prává, o kousek dál tmou svítí Velký vůz.
} \refsm{}
