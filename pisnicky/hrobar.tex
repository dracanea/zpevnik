\chord{F}kopat hroby byl můj ide\chord{G}ál.
}

\vers{2}{
\chord{D}Jezdit s hlínou, jezdit s vozíkem, \chord{Hm}s černou rakví, s bílým pomníkem,\\
\chord{G}toho bych se nikdy nena\chord{A\7}dál.

\vers{3}{

\vers{4}{

marně skřípěj kola brzdící.
}

\vers{5}{
Stoupám vzhůru, stoupám ke hvězdám, tam se s černou rakví neshledám,\\
sbohem moje město zářící.
}
